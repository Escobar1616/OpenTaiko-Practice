## Feature

### 便利系
* ログイン演出等をスキップ
* Boot時に選曲画面に移行
* Thank you画面を無効化
* デバッグ情報にオーディオ出力情報とオーディオ遅延を表示

### 練習系
* 特訓モードで往復練習しやすいように、開始地点が巻き戻るように
* ms単位で入力のズレを表示 (OFF可能、ギタドラ風の表示、FAST/SLOWのみ表示も可能)(可/不可時のみ表示設定可能)
